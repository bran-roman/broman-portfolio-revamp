This folder contains a sample Python application named "Blue Bird Banking" which acts as an example of a checkbook bank system program. Users can create accounts by inputting their information along with their any initial deposits they'd like to submit. Once created, they may use the dropdown menu to search for their account and view their recent transaction, as well as their current balance.
